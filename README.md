# Foro Duoc sede Maipu
## Descripción
Web diseñada para el proceso quirúrgico del servicio de salud metropolitano sur oriente (SSMSO).

A utilizarse como intranet corporativa de los hospitales adheridos al SSMSO.

## Tecnologías
### Backend
- Django
### Frontend
- HTML5
    - HTML
    - CSS3
    - JavaScript
- Bootstrap
